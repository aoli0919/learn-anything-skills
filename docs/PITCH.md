# Project Pitch

## One Sentence

Learn Anything Skills is a beginner-first agent skill pack that turns a vague learning goal into a map, tutor, project path, and feedback loop.

## Short Pitch

Most AI tutors are good at answering questions, but beginners often do not know what to ask.

Learn Anything Skills solves the earlier problem: it helps a learner choose the next right question, turn it into a task, test understanding, build a small artifact, and remember what happened.

It is especially useful for fields like embodied AI, where the beginner path crosses concepts, papers, code, projects, and physical intuition.

## README First Screen Promise

You bring:

- a topic
- your current background
- a rough time budget

The skills return:

- a 30-day path
- a beginner field map
- active tutoring checks
- project cards
- paper-to-practice notes
- daily reflection memory

## Star Conversion Risks

- The project looks like prompts only and not a system.
- The README is too abstract.
- No realistic examples.
- Installation is unclear.
- The embodied AI example overpromises.
- The skills do not produce visible artifacts.
- The project lacks validation.

## How This Repo Addresses Them

- Fourteen separate skills with clear responsibilities.
- Hot-pattern extensions for trend scans, source curation, book-to-skill conversion, codebase courses, knowledge graphs, stuck diagnosis, exam simulation, and token-frugal mentoring.
- Install script and validation script.
- Embodied AI examples.
- Templates for logs, concepts, and projects.
- Honest language about difficulty.
- Beginner-first project defaults.
