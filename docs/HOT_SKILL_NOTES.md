# Hot Skill Notes

These notes record the patterns used when adding the eight extension skills.

## Patterns Observed

Popular agent skill repositories tend to share these traits:

- one clear trigger per skill
- small composable capabilities
- examples near the README
- optional scripts, templates, or metadata
- strict output formats
- safety around source quality, resource caps, and validation
- a clear "what to ignore" section
- a visible artifact, not only advice

## How The New Skills Map To Those Patterns

- `token-frugal-mentor`: concise mode and low-token interaction.
- `source-scout`: resource curation instead of resource dumping.
- `trend-radar`: learning from fast-moving topics without doomscrolling.
- `book-to-skill`: source-to-skill conversion.
- `codebase-apprentice`: codebase-to-course learning.
- `knowledge-graph`: graph-style concept memory.
- `stuck-debugger`: diagnosis and recovery loop.
- `exam-simulator`: validation through tests and rubrics.
