# Hot Skill Extensions Gallery

These examples show the eight added skills inspired by popular skill patterns: concise mode, source curation, source-to-skill conversion, codebase-to-course learning, knowledge maps, stuck diagnosis, testing, and trend scanning.

## 1. Trend Radar

```text
$trend-radar What changed in embodied AI this week?
```

Expected artifact:

- weekly radar table
- top signals
- hype to ignore
- today's learning action

## 2. Source Scout

```text
$source-scout Find 3-5 beginner resources for robot learning.
```

Expected artifact:

- ranked resource stack
- what to use each source for
- what to skip
- first task

## 3. Book To Skill

```text
$book-to-skill Turn this robotics course into a study skill.
```

Expected artifact:

- chapter map
- must-study / skim / later
- mini SKILL.md outline
- first 90-minute session

## 4. Codebase Apprentice

```text
$codebase-apprentice Teach me this repository as a beginner course.
```

Expected artifact:

- codebase map
- reading path
- trace task
- 1-hour modification task

## 5. Knowledge Graph

```text
$knowledge-graph Map observation, action, policy, reward, and simulation.
```

Expected artifact:

- nodes
- edges
- weak nodes
- review cards
- next edge to study

## 6. Stuck Debugger

```text
$stuck-debugger I keep reading papers but nothing sticks.
```

Expected artifact:

- blocker diagnosis
- task downgrade
- 30-minute recovery plan
- resume point

## 7. Exam Simulator

```text
$exam-simulator Test me on embodied AI basics.
```

Expected artifact:

- recall/application/transfer questions
- answer key
- rubric
- review plan

## 8. Token Frugal Mentor

```text
$token-frugal-mentor I am overwhelmed. Give me only the next useful step.
```

Expected artifact:

```text
Goal:
Today:
Do:
Done when:
Check:
Ignore:
Stop after:
Log:
```
