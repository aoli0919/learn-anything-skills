# Example: AI Paper Reading Loop

## Input

```text
$paper-to-practice I want to read my first AI paper without getting lost.
I know basic ML, but methods sections feel dense.
```

## Output Sketch

### Reading Target

Pick one paper only. Do not open a survey yet.

### Task Setup

Before reading the method, fill this table:

| Question | Answer |
| --- | --- |
| What is the input? | |
| What is the output? | |
| What is the dataset or environment? | |
| What is the baseline? | |
| What is the evaluation? | |

### Method Cards

Create 3 cards:

1. Main model component
2. Training signal
3. Evaluation metric

### 2-Hour Toy Reproduction

Do not reproduce the full paper.

Reproduce one tiny idea:

- a simplified dataset
- a smaller model
- one metric
- one result table

### Check Question

If you removed the paper's main component, what should get worse and why?

### Reflection

Log whether the paper was hard because of vocabulary, math, code, or missing background.
